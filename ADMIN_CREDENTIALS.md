## Admin Panel Access

The admin panel is available at the `/admin` route in your application.

### Features Available in Admin Panel:

1. **Bookings Manager** - View and manage all customer bookings
2. **Services Manager** - Add, edit, and manage services
3. **Images Manager** - Manage hero section images
4. **Settings Manager** - Configure salon settings
5. **Dashboard** - View key metrics and statistics

## Important Security Notes

- Keep these credentials secure
- Change the password after first login if deploying to production
- Admin panel requires authentication to access
- All admin actions are logged in the system

## Access URL

Once the application is running, access the admin panel at:
```
http://localhost:5173/admin
```

Or on your deployed domain:
```
https://yourdomain.com/admin
```

## First Time Setup

1. Navigate to the admin login page
2. Enter the credentials above
3. Click "Sign In"
4. You'll be redirected to the admin dashboard
