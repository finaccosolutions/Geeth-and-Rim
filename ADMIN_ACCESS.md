# Admin Panel Access

## Admin Credentials

**URL:** `/admin` (or `http://localhost:5173/admin` in development)

**Email:** `admin@geetandrim.com`

**Password:** `Admin@12345`

## Important Notes

- These are the default admin credentials created for the Geetandrim Salon website
- The admin panel provides access to manage:
  - Customer bookings
  - Services and service categories
  - Hero and gallery images
  - Email settings (SMTP configuration)

- For production deployment, it is highly recommended to change these credentials immediately after first login

## First Time Login

1. Navigate to the `/admin` route
2. Enter the email and password above
3. Click "Sign In"
4. You will be redirected to the admin dashboard

## Security Reminder

Keep these credentials secure and change them before deploying to production.
